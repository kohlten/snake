# snake

The Snake game written in C using SDL2.

To compile you need:
	make
	gcc
	Linux:
		X11
		Xext
		Xcursor
		asound
		sndio
		cairo
		wayland
		pulseaudio
		freetype
	Mac:
		freetype
		X11


After getting the dependencies, simply run sh install.sh.
If there's any issues it should let you know, if not please contact me for help.
